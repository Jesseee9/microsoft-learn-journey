
# Microsoft Entra ID – Learning Path

## Overview
Completed the Microsoft Learn learning path focused on Microsoft Entra ID, Microsoft’s cloud-based identity and access management (IAM) solution.

This learning path covered how Entra ID helps organizations manage identities, control access to resources, and secure users, devices, and applications in modern cloud environments.

## Skills Learned
- Understanding what Microsoft Entra ID is and its role in identity and access management  
- Core identity concepts: users, groups, and directory services  
- Authentication vs authorization and why both matter  
- Managing access to applications and cloud resources  
- Basics of single sign-on (SSO) and identity federation  
- Role of Entra ID in securing cloud and enterprise environments  
- How Entra ID integrates with services like Microsoft Intune and Azure  

## Key Takeaway
Microsoft Entra ID acts as the identity backbone of modern IT infrastructure, enabling secure access, centralized identity management, and seamless integration across cloud services.
